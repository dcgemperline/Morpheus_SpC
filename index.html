MSpC

Morpheus Spectral Counter or MSpC

This is a an application for Spectral Count Based Label-free Quantitative (LFQ) proteomics that integrates with the Morpheus Mass Spectrometry Search Engine. It is developed in C# and is open sourced under an MIT License.

To use this software your data must be searched with the Morpheus Mass Spectrometry Search Engine. The Morpheus Mass Spectrometry Search Engine is written by Craig Wenger and is available here:

http://sourceforge.net/projects/morpheus-ms/
